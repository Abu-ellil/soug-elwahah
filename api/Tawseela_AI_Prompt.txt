
أنت مهندس برمجيات متمرس، مطلوب منك توليد مشروع **Tawseela Backend** كامل كما لو أنك تنفذه لفريق تطوير محترف.

### 1️⃣ Backend
- Node.js + Express
- Modular architecture: modules/auth, modules/users, modules/drivers, modules/stores, modules/orders, modules/payments, modules/notifications, modules/admin
- Socket.io للتتبع والإشعارات
- JWT Auth (Email/Password فقط)
- Logger + Error Handling
- Node-cron للمهام المجدولة

### 2️⃣ Database (MongoDB Atlas)
- Users: role ['customer','driver','store_owner','admin'], phone, email, password, location (GeoJSON), wallet, isVerified, isActive, driverDocuments {nationalId, license, photo, status: ['pending','approved','rejected']}
- Stores: name, owner, products, workingHours, zone
- Products: name, price, stock, image, store
- Orders: customer, store, driver, items [{product, quantity, price}], status ['pending','confirmed','preparing','on_way','delivered','cancelled'], payment {method, status, amount}, deliveryFee, totalAmount, location {pickup, delivery}, timestamps
- Drivers: verification documents, status
- Transactions: type, amount, user, order, status
- Notifications: user, type, message, read, timestamps
- Zones: name, coordinates, active

### 3️⃣ Features
- Driver verification system (رفع بطاقة + صورة + مراجعة الأدمن)
- Real-time GPS tracking لكل سائق باستخدام Socket.io
- Orders CRUD مع تحديث الحالة وتتبع التسليم
- Payments: محاكاة Paymob/Fawry sandbox endpoints
- Admin Dashboard: مراجعة السائقين، إدارة الطلبات، إعدادات العمولة
