# تطبيق التاجر

هذا التطبيق مخصص للتجار الذين يرغبون في إدارة متاجرهم ومنتجاتهم. يتيح للتجار إنشاء حساب، وانتظار موافقة الإدارة، وإدارة منتجاتهم ومتجرهم.

## الميزات

- تسجيل تاجر جديد مع معلومات المتجر
- انتظار موافقة الإدارة على الحساب
- إدارة المنتجات (إضافة، تعديل، حذف)
- تعديل معلومات المتجر (الاسم، الوصف، الصورة)
- لوحة تحكم تفاعلية مع إحصائيات المبيعات

## الميزات الإدارية

- واجهة إدارة طلبات تسجيل التجار
- إمكانية قبول أو رفض طلبات التسجيل

## التقنيات المستخدمة

- Expo
- React Native
- TypeScript
- Expo Router (نظام التوجيه)
- AsyncStorage (لتخزين البيانات محليًا)
- React Navigation
- Expo Image Picker

## كيفية التثبيت

1. تأكد من تثبيت Node.js و Expo CLI على جهازك
2. نسخ المشروع:

```bash
git clone <repository-url>
```

3. الانتقال إلى مجلد التطبيق:

```bash
cd merchant-app
```

4. تثبيت الحزم:

```bash
npm install
```

5. تشغيل التطبيق:

```bash
npx expo start
```

## ملاحظات

- هذا الإصدار يستخدم AsyncStorage لتخزين البيانات محليًا. في الإصدار الإنتاجي، يجب ربطه بخادم حقيقي.
- تم دعم اللغة العربية بالكامل في واجهة المستخدم.
- يدعم التطبيق التسجيل، تسجيل الدخول، إدارة المنتجات، وتعديل معلومات المتجر.

## بنية المجلدات

```
merchant-app/
├── app/
│   ├── (auth)/          # شاشات المصادقة (تسجيل الدخول، التسجيل)
│   ├── (tabs)/          # شاشات التطبيق الرئيسية
│   │   ├── index.tsx    # الشاشة الرئيسية
│   │   ├── products.tsx # إدارة المنتجات
│   │   ├── profile.tsx  # الملف الشخصي وتعديل المتجر
│   │   └── _layout.tsx  # تخطيط علامات التبويب
│   └── admin.tsx        # واجهة الإدارة
├── contexts/            # سياقات React (مثل المصادقة)
├── components/          # مكونات واجهة المستخدم
└── assets/              # الملفات الثابتة
```
